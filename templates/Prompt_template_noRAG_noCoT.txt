## Task

You are a highly knowledgeable clinical trial design expert specializing in Phase 3 trial protocol development. Your task is to design a comprehensive and scientifically rigorous set of **eligibility criteria** for the following drug-related Phase 3 clinical trial. This is a crucial component that will directly impact patient safety, trial validity, regulatory approval potential, and eventual clinical adoption.

## Instructions

Standard (Balanced Approach): 
- Objective: Balance scientific validity, patient safety, and recruitment feasibility.
- Typical profile: Clinically justified thresholds; excludes known high-risk groups; allows common comorbidities with controls.

## Eligibility Criteria Requirements

- Include both **Inclusion** and **Exclusion** Criteria.
- Inclusion Criteria should define the patient population that is eligible to participate in the trial.
- Exclusion Criteria should identify characteristics or conditions that would disqualify a patient from participation in the trial.
- Ensure your output is professional, precise, unambiguous, and employs appropriate medical terminology.
- Avoid drafting repetitive criterion related to the same variable in inclusion and exclusion criteria. For instance:
  - If the inclusion criteria specify participants aged 18 and above, don't repeat excluding those under 18 in the exclusion criteria. 
  - If "unilateral" is set, no need to mention "bilateral/multilateral" in exclusion. 
  - If "symptomatic" is set, no need to mention "asymptomatic" in exclusion, and so on.

## Input (Clinical trial title)

{trial_title}

## Output (Directly output the eligibility criteria using the following JSON format)

{{
  "InclusionCriteria": [
    {{
      "Criterion": "A specific, measurable, and clinically appropriate inclusion criterion."
    }}
  ],
  "ExclusionCriteria": [
    {{
      "Criterion": "A specific, measurable, and clinically appropriate exclusion criterion."
    }}
  ]
}}
