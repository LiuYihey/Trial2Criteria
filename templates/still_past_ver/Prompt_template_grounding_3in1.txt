## Eligibility Criteria Requirements

### Critical Considerations
1. **Safety Considerations**: Carefully evaluate risks based on drug mechanism, known adverse effects, and target population vulnerabilities.
2. **Scientific Validity**: Ensure criteria support robust hypothesis testing and minimize confounding variables.
3. **Statistical Power**: Consider how criteria affect sample size, recruitment feasibility, and study generalizability.
4. **Ethical Standards**: Balance scientific rigor with inclusivity, avoiding unnecessary exclusions of demographic groups.
5. **Regulatory Alignment**: Align criteria with FDA/EMA guidelines for Phase 3 trials of similar interventions.

### Formatting Instructions
1. Include both **Inclusion** and **Exclusion** Criteria with clear categorization.
2. Use precise, measurable parameters.
3. Organize criteria logically by systems or clinical relevance.
4. Consider addressing these key categories:
   - Demographics (age, sex, reproductive status)
   - Disease characteristics (diagnosis, severity, duration, staging)
   - Laboratory values and test results
   - Prior/concomitant treatments
   - Comorbidities
   - Safety parameters
   - Practical/logistical considerations

### Set-Specific Instructions (what differs across the three sets)

1) Standard Criteria Set (Balanced Approach)
- Objective: Balance scientific validity, patient safety, and recruitment feasibility.
- Typical profile: Clinically justified thresholds; excludes known high-risk groups; allows common comorbidities with controls.
- Trade-offs: Moderate inclusivity; maintain core safety labs/vitals and disease-definition rigor.

2) Broader Criteria Set (Increased Generalizability)
- Objective: Maximize real-world applicability and diversity while maintaining essential safety.
- Typical profile: Fewer exclusions; wider ranges (age, labs) when clinically acceptable; allow stable comorbidities.
- Required justification: For broadened items, explain safety controls and monitoring in Rationale; avoid compromising primary outcome integrity.

3) Stringent Criteria Set (Maximized Internal Validity)
- Objective: Reduce confounding and maximize signal detection for confirmatory efficacy.
- Typical profile: Narrower disease definitions/staging; stricter lab/vital thresholds; limit concomitant therapies; require adherence predictors.
- Required justification: Explain how restriction improves endpoint precision and reduces variability/confounding.

### Evidence Grading System
For each criterion you develop, classify the supporting evidence using this system:
- **High Certainty [H]**: Multiple consistent sources support this criterion (e.g., multiple Phase 3 trials, strong drug mechanism evidence, clear disease guidelines)
- **Moderate Certainty [M]**: Limited but clear evidence supports this criterion (e.g., consistent Phase 2 data, established clinical practice)
- **Low Certainty [L]**: Theoretical basis or extrapolated evidence supports this criterion (e.g., mechanism-based reasoning, limited clinical data)
